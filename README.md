# Food Delivery Risk Analysis 🚀

## 📌 Project Overview
This project analyzes revenue leakage in food delivery systems by identifying fraud patterns, wastage, and order trends using data analysis and time series modeling.

---

## 🎯 Objectives
- Detect fraudulent behavior in orders
- Analyze food wastage due to cancellations
- Forecast order demand using LSTM
- Compare classical and advanced models

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- Machine Learning (LSTM - TensorFlow/Keras)
- SQL (for analysis)
- Excel (dataset + dashboard)
- Google Colab

---

## 📊 Dataset Features
- Order Date
- Total Orders
- Total Cancellations
- Wastage Flag
- Fraud Flag

---

## 🤖 Models Used
- LSTM (Advanced Model)
- Moving Average (Baseline Model)

---

## 📈 Results
- LSTM Model MAPE ≈ ~8-10%
- Moving Average performed slightly better due to small dataset
- Demonstrates importance of model selection

---

## 🧠 Key Insights
- High cancellations lead to wastage
- Fraud patterns observed during spikes
- Model performance depends on data complexity
